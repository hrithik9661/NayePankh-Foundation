NayePankh Foundation – NGO Website

About the Project
NayePankh Foundation is a responsive NGO website designed to represent social initiatives focused on education, skill development, and community empowerment. The website provides a structured and user-friendly interface to showcase the organization’s work and impact.

Features
Responsive landing page design
Hero section with background image
About section with mission, vision, and values
Impact statistics section with counters
Programs section highlighting activities
Testimonials section for success stories
Image gallery with lightbox functionality
Volunteer registration form
Dark and light mode toggle
Mobile responsive layout

Tech Stack
HTML5
CSS3 (Flexbox and Grid)
JavaScript (DOM manipulation)
Font Awesome Icons
Google Fonts

Purpose
This project demonstrates the use of frontend web development to build a structured and professional NGO website. It focuses on UI design, responsiveness, and interactive elements.

How to Run

Clone the repository
Open index.html in any browser
